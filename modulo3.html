¿Qué es un archivo?
La diferencia entre archivos y carpetas

Un archivo es cualquier cosa que puedes abrir y usar: una carta que escribiste, una foto, una canción, un vídeo, un PDF...

La diferencia con una carpeta:

📁
CARPETA

Es un contenedor. No tiene contenido propio, pero dentro puedes guardar archivos y otras carpetas. No puedes "abrirla" para ver texto o imágenes, solo para ver lo que hay dentro.

📄
ARCHIVO

Tiene contenido real: texto, una imagen, música... Al hacer doble clic, se abre con el programa adecuado y puedes ver o usar ese contenido.

Tipos de archivos más comunes
Cada tipo de archivo tiene su propio icono y extensión

Los archivos tienen un nombre y una extensión. La extensión (las letras después del punto) indica el tipo de archivo que es.

Ejemplo: carta.docx → el nombre es "carta" y la extensión ".docx" indica que es un documento de Word.

Abrir un archivo
Doble clic es todo lo que necesitas

Para abrir un archivo, simplemente haz doble clic sobre él. Windows abrirá automáticamente el programa adecuado.

Por ejemplo: si haces doble clic en una foto, se abre el visor de fotos. Si haces doble clic en un documento Word, se abre Microsoft Word.

💡 ¿Y si el archivo no se abre bien? Haz clic derecho sobre el archivo y elige "Abrir con". Podrás elegir qué programa quieres usar para abrirlo.

Guardar un archivo
Muy importante: guardar el trabajo antes de cerrar

Guardar significa grabar los cambios que has hecho en un archivo para que no se pierdan cuando cierras el programa.

Si no guardas, todo lo que hayas escrito o cambiado desaparece cuando cierras el programa.

Copiar y pegar archivos
Las mismas teclas que para las carpetas

Copiar y mover archivos funciona exactamente igual que con las carpetas. Ya lo aprendiste en el Módulo 2, así que esto te será fácil.

Acción Teclas Qué hace
Copiar Ctrl + C El archivo queda en los dos sitios
Cortar (mover) Ctrl + X El archivo se mueve, desaparece del sitio original
Pegar Ctrl + V El archivo aparece en el nuevo sitio
Deshacer Ctrl + Z ¡Deshace lo último que hiciste! Muy útil si te equivocas
💡 El mejor truco del teclado: Ctrl+Z = Deshacer. Si hiciste algo mal (borraste algo, moviste algo sin querer...), pulsa Ctrl+Z y vuelve al estado anterior. ¡Como una máquina del tiempo!
